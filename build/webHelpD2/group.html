group

Group columns into ColumnsGroups.

It is a special case of move operation.

group { columns } .into(groupName) | .into { groupNameExpression } groupNameExpression = DataColumn.(DataColumn) -> String
df.group { age and city }.into("info") df.group { all() }.into { it.type().toString() }.print()

To ungroup grouped columns use ungroup operation.
