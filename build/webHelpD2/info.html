Dataframe 1.0 Help

General info

General information about DataFrame:

  • nrow()- number of rows

  • ncol()- number of columns

  • columnNames()- list of column names

  • head(n)- first n rows (default 5)

  • schema()- schema of columns

  • describe()- general statistics for every column
